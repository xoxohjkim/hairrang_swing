# 헤어랑 - 미용실 영업 관리 프로그램

## 메뉴 구성 
영업관리
- 고객 관리(수정 및 삭제)
- 고객 검색

주문관리 
- 헤어 주문
- 헤어 주문 내역

영업현황
- 주문 통계

# 요구사항 
- 프로그램 시작 시 고객 목록을 출력한다. 
- 고객의 정보를 입력받아 등록할 수 있다.
- 고객 항목을 클릭 시 고객의 정보를 수정, 삭제할 수 있다.
- 고객의 이용내역을 조회할 수 있다.
- 고객 항목을 선택 후 주문 프로그램으로 이동 시 고객명, 고객번호가 자동입력된다.
- 헤어명 선택 후 이벤트명을 선택하면 해당 이벤트의 할인율이 자동 적용된다.
- 기간 별 주문 내역을 조회할 수 있다.
- 주문 통계를 년도별, 월별로 조회할 수 있다.
